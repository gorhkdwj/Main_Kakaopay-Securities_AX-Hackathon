> **제출처**: arena.jocodingax.ai (전화번호+이름 로그인, 12:00~13:00). 제출물 = 이 브리프 + 프롬프트 로그.
> **작성**: 2026-07-18 (본선 당일 새벽 초안). 실제 문제 공개(10:00) 후 `[당일 반영]` 표시 부분을 갱신.
> **정합 원칙**: 아래 모든 기능 주장은 하단 §대조표의 구현 증거와 1:1로 연결된다. 효과(불안 감소 등)는 **검증 전 가설**로만 말하고, "검증된" 같은 단정은 쓰지 않는다.

## ① 한 줄 요약

초보 투자자가 **매수·매도·보류 어느 판단 순간에든**, 출처·기준시각이 붙은 판단 재료를 구조화해 보고 **스스로 설명 가능한 결정**을 내리도록 돕는 "판단 전 브리핑" 데모. **AI가 방향 결론(사라/팔라)을 말하지 않는 것**이 핵심 경계다.

## ② 문제 정의

> `[당일 반영]` — 실제 문제문에 맞춰 아래 프레임을 조정.

- 초보 투자자는 매수·매도 판단이 흔들리는 순간, 특히 **급락·손실 같은 고불안 순간**에 판단 왜곡이 가장 크다(리서치 근거).
- 시중 도구는 '지금 사세요/파세요' 같은 방향 결론이나 수익 예측을 제시해 왜곡을 키우거나, 근거 없는 커뮤니티 분위기에 휩쓸리게 한다.
- 트랙 문제 방향("초보 투자자를 위한 주식 매수/매도 지원")과의 정합: 우리는 '지원'을 **방향 제시가 아니라, 스스로 판단할 재료와 선택권의 제공**으로 해석했다.
- 대표 시연 장면: −8% 급락한 보유 종목(가온전자) 앞에서의 매도 충동 순간.

## ③ 핵심 기능·시연 포인트

> `[당일 반영]` — 시연 동선은 문제에 맞게 조정 가능(기본 동선은 아래).

**흐름**: S0(실앱 주문 화면 재현) → ① 상황 → ② AI 브리핑 → ④ 시나리오 비교 → ⑤ 검토 의향 → ⑥ 모의 체결 → ⑦ 투자 일지 → ⑧ 종착.

**기능 = 시연 포인트**

1. **출처·기준시각 배지** — 모든 사실에 출처 ID·기준시각을 붙이고, 출처 없는 사실은 화면에 나오기 전 **차단**한다.
2. **사실 / 해석 / 모름 분리** — AI 응답을 facts(출처 있음)·interpretations(긍정·부정 양면)·unknowns(구체적으로 "언제 확인되는지" 병기)로 구조화한다.
3. **대칭 선택지** — 매수·매도·보류를 기본 강조 없이 대칭으로 제시(유도 없음). **양방향 흐름**이라 급락 시 저가 매수 검토도 동일 구조로 성립한다.
4. **사전 고지 + 별도 화면 모의 체결** — 수수료·세금·D+2 정산·비가역성을 체결 **전에** 고지하고, 실제 체결이 아닌 **모의**임을 상시 표기한다.
5. **계산·생성 분리** — 체결·비용·세금·정산·잔여 비중·평단가 등 **숫자는 결정론 계산 엔진**이 만들고, LLM은 **근거가 붙은 설명만** 한다(LLM이 산수하지 않음).
6. **안전 게이트** — 출처 없는 사실 0·권유/단정 표현 0·기준시각 누락 0을 자동 검사하고 **카운터를 화면에 표시**한다. 프롬프트 주입은 자료 격리 + 최종 가드로 방어한다.
7. **장애 안전모드 / 오프라인 완주** — 네트워크·LLM 실패 시 캐시·정적 조립으로 흐름을 **완주**한다(외부 자산 0, 네트워크 0 경로 존재).

**시연 동선**: `loss8`(가온전자 −8%) 완주 → 시나리오 전환 1회(`profit15` 한빛식품 +15% 평가 / `first_buy` 다온소재 탐색 매수)로 "급락 전용이 아닌 범용 판단 구조"임을 화면으로 증명. 시간 압박 시 전환은 "질문 시"로 강등.

> 효과(초보의 불안 감소·상담 문의 감소 등)는 **검증 전 가설(H/M/L)**이며, 이 데모는 효과를 주장하지 않는다 — 구조와 경계를 보여줄 뿐이다.

## ④ 사용 도구·데이터 출처

- **개발 도구**: Claude Code · Codex CLI (전 과정 프롬프트 로그 수집·제출).
- **기술 스택**: FastAPI + vanilla JS(외부 CDN·웹폰트·원격 이미지 0 → 완전 오프라인 동작), httpx(REST 직행, 별도 SDK 없음).
- **AI 모델**: Anthropic Claude API(`claude-sonnet-5`) — ② 브리핑의 **'설명'만** 생성하며 **숫자는 생성하지 않는다**. 응답 원천은 3계층 폴백(live → cache → static)이고, 어느 원천이든 렌더 전 동일한 정책 가드를 통과한 블록만 화면에 나온다.
- **데이터**: 가상 종목 fixture 3종(가온전자 DEMO-GAON · 한빛식품 DEMO-HANBIT · 다온소재 DEMO-DAON — **실존 종목 아님**, 권유 오해 방지). 사실 카드는 가상 공시·시세이며 '교육용 가상 데이터' 배지로 명시한다.
- **근거**: 리서치 교차검증 완료본(출처 402건·주장 123건, `docs/report/`).

## 선택 항목 — 한계·미구현 (정직 고지)

- **시장 지수 실데이터 스냅샷 수집(S6)은 미실행** — 현재 시나리오는 fixture 단독이며 '교육용 가상 데이터' 배지로 명시한다.
- **live LLM 브리핑은 네트워크 의존** — 오프라인/실패 시 캐시·정적으로 자동 강등되고, 그 원천을 화면 배지로 표시한다.
- 일부 수치는 `[데모 고정]`값(계약 §13)으로 실서비스 전 재검토 대상이다.
- 실주문·개인화 종목 추천·수익 예측·게임화는 **의도적으로 미구현**(제품 경계 — 리서치 확정 §14).
- 데모 URL: 로컬(`http://127.0.0.1:8765`), 공개 배포 없음.
