# 본선 문제 공개 완료



주식 매수/매도 지원 시스템
배경
투자자들이 가장 알고 싶어 하는 것은 "어떤 주식을 사야 할까?(What to buy)"입니다.

하지만 종목을 정하고 나면 곧바로 "이 주식을 어떻게 사고팔아야 할까?(How to trade)"라는 현실적인 난관에 부딪힙니다.

삼성전자를 1,000만 원어치 매수하기로 했을 때, 한 번에 다 사야 할지 나누어 사야 할지 판단하기는 쉽지 않습니다.

수익 실현·손절의 순간도 마찬가지입니다.

투자자의 마지막 실행 단계를 스마트하게 돕는 '주식 매수/매도 지원 시스템'을 개발하세요.

타겟 사용자
초보 투자자 — 종목 선정까지는 마쳤으나 호가창 앞에서 매수/매도 타이밍이 막막한 분들을 위한 직관적이고 친절한 가이드.

데이터 (제약)
증권사 내부 데이터는 사용할 수 없습니다.

야후 파이낸스 등 외부 오픈 API, 합법적 범위의 웹 크롤링으로 수집한 퍼블릭 데이터를 활용하세요.

평가 기준 (4항목)

1. 접근의 참신성 — 초보 투자자의 막막함을 푸는 방식이 얼마나 창의적인가
2. UI/UX 및 사용자 Flow — 초보 투자자가 복잡함 없이 매끄럽게 쓸 수 있는가
3. 기술 완성도 — AI 모델 활용·데이터 파이프라인이 안정적이고 논리적인가
4. 실제 적용 가능성 — 실제 서비스 연동·비즈니스 임팩트를 낼 수준인가
